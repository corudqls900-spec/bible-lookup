🔑 성경 API 설정
무료 API 키 1분 설정 후 66권 전체 한국어 성경을 바로 사용할 수 있습니다.
→ scripture.api.bible 바로가기 ↗
  1. scripture.api.bible 접속 → Get Started Free 클릭
  2. 이메일로 가입 후 로그인 → My Apps → + Add App
  3. 앱 이름 아무거나 입력 후 생성 → API Key 복사해서 아래에 붙여넣기
